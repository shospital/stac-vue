About PolarWatch

PolarWatch is a thematic node within the NOAA CoastWatch program and supports the goals set forth in NOAA’s Arctic Vision and Strategy.

PolarWatch provides access to satellite observation and information products for the high latitude regions, including coastal and offshore Alaska, the Arctic and Southern oceans.
